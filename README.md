Installation:

1. Clone the repository and navigate to the project folder:
   ```bash
   git clone
   ```

2. Install pnpm globally:
    ```bash
    npm install -g pnpm
    ```

3. Install project dependencies:
    ```bash
    pnpm install
    ```

4. Run the project:
    ```bash
    pnpm run dev
    ```

5. Component tests:
    ```bash
    pnpm test
    ```

6. E2E tests:
    ```bash
    pnpm test:e2e
    ```
